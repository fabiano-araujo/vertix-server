AbacatePay - Integração Pix

Bem-vindo(a) ao Demo de Integração Pix!

Esta aplicação demonstra como integrar pagamentos via Pix usando a API do AbacatePay. Siga os passos abaixo para testar a integração:

  1. Gere um token de teste na página "Obter Token"
  2. Vá para a página de "Teste de Pagamento Pix"
  3. Cole o token gerado no campo apropriado
  4. Preencha os detalhes do pagamento e gere uma cobrança Pix
  5. Escaneie o QR Code gerado ou copie o código Pix para simular o pagamento
